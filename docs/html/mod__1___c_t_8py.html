Detailed Description

This file contains code for the segmentation of the CT scan image.

@detail The lungs are segmented using a region growing algorithm. The seeds for the region growing are the 2 smallest regions in the thresholded image. The image is thresholded using Otsu's method, and closing is applied to remove small objects.
