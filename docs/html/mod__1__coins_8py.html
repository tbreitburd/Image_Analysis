Detailed Description

This module contains code for the segmentation of the coins.

This module contains the code for the segmentation of the coins. This is done by first removing the "corruption" lines using inpainting, then increasing the contrast of the image, removing the background using a rolling ball algorithm, and finally segmenting the coins using K-means clustering. The coins are then labelled and the results are plotted.
